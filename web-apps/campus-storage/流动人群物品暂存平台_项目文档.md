## 一、可行性分析报告

### 1.1 市场背景与用户规模

#### 目标用户群体

| 用户群体 | 规模数据 | 核心需求 | 频率 |
|---------|---------|---------|------|
| **农民工** | 2.99亿人（2024年国家统计局），其中外出农民工1.79亿人 | 工具/大件物品无法随身携带上火车；目的地不确定 | 季节性流动（春耕、秋收、春节前后） |
| **求职者/实习生** | 每年约900万高校毕业生 + 职场跳槽人群 | 租房未确定、面试地点多变、行李寄存 | 毕业季（6-8月）、求职高峰期 |
| **跨城工作者** | 新增需求群体 | 工作调动、临时出差行李寄存 | 不定期 |

#### 痛点验证

**农民工群体痛点：**
- 火车行李限制：普通旅客免费携带行李上限仅 **20公斤**
- 工具禁带：斧子、锤子、钢锉、木工刨、钻机、焊枪等工具 **禁止随身携带**，必须托运
- 目的不确定：往往只知道去某个省份/城市，具体工作地点和住宿地址未定
- 现有方案缺陷：传统物流需要确定收货地址，无法满足"先寄后定"需求

**求职者/实习生痛点：**
- 租房未确定：面试期间住宿地址未定，行李无处安放
- 时间紧迫：毕业季需快速离校，但新住处未落实
- 现有方案：快递需确定地址，迷你仓价格较高且需自行搬运

### 1.2 竞品分析

| 竞品 | 服务模式 | 优势 | 劣势 | 差异化机会 |
|------|---------|------|------|-----------|
| **菜鸟驿站** | 快递代收+暂存 | 覆盖广（17万+站点）、免费暂存 | 需确定收货地址、暂存时间短 | 不支持"地址未定"场景 |
| **丰巢智能柜** | 自助快递柜 | 24小时自助、覆盖广 | 超时收费、柜体容量有限 | 不支持大件物品、跨城服务弱 |
| **智惠存/迷你仓** | 仓储服务 | 长期存储、大件支持 | 价格高、需自行搬运 | 不提供物流配送 |
| **京东/申通"毕业寄"** | 校园行李寄递 | 一站式服务、优惠价格 | 需确定收货地址 | 仅针对毕业生，季节性强 |
| **行李寄存平台** | 车站/景点寄存 | 临时寄存便捷 | 按天计费贵、无配送服务 | 不支持跨城+配送闭环 |

### 1.3 市场空白与机会

**核心差异化定位：**
```
"地址未定，物品先行" —— 面向流动人群的物品暂存+配送一体化服务
```

**市场空白点：**
1. **无确定地址寄送**：现有物流均需确定收货地址，本项目填补空白
2. **大件/工具物品**：快递柜无法容纳，本项目支持大件仓储
3. **跨城闭环服务**：暂存+配送一体化，用户无需多次操作
4. **下沉市场触达**：农民工群体数字化程度低，需线下+线上结合

### 1.4 商业模式设计

> **v1.1 更新说明**：商业模式从"服务费为主"调整为"广告收入为主"，大幅降低用户端价格，以低价/免费服务快速获取用户规模，通过广告变现实现盈利。

#### 核心商业逻辑

```
低价/免费服务 → 快速获取大量流动用户 → 高频使用场景 → 广告精准投放 → 广告主付费
```

**战略思路**：流动人群（农民工、求职者）是广告主高度关注的消费群体。通过低价暂存服务吸引大量用户，平台成为连接品牌与流动人群的精准广告渠道。

#### 收入来源

| 收入类型 | 定价策略 | 预估占比 | 说明 |
|---------|---------|---------|------|
| **广告收入（核心）** | CPM/CPC/CPA 多种模式 | **60%** | 主盈利来源，详见下方广告体系 |
| **暂存服务费** | 低价策略（例：0.5m³/3-5元/天） | 15% | 大幅降低，覆盖基础运营成本 |
| **配送服务费** | 微利定价（按距离+重量） | 10% | 与合作平台分成 |
| **保价服务费** | 物品价值的1-3% | 5% | 增值收入 |
| **增值服务** | 打包、代收、拍照存档等 | 5% | 补充收入 |
| **数据服务** | 匿名化用户洞察报告 | 5% | 面向 B 端客户 |

#### 广告体系设计

**广告位规划：**

| 广告位 | 位置 | 形式 | 目标广告主 |
|--------|------|------|-----------|
| **开屏广告** | 小程序启动页 | 静态图片/短视频 3-5s | 品牌曝光（快消品、手机品牌） |
| **首页 Banner** | 首页顶部/中部 | 轮播图片 | 本地生活服务、招工信息 |
| **订单流程广告** | 下单成功页、状态查询页 | 原生信息流 | 保险、金融、通信套餐 |
| **消息推送广告** | 微信服务通知 | 文字+链接 | 招工平台、技能培训 |
| **仓库场景广告** | 合作仓库线下展示 | 海报/易拉宝 | 工具品牌、劳保用品 |
| **配送包裹广告** | 包裹贴纸/面单背面 | 印刷广告 | 快消品试用装派发 |

**广告主画像：**

| 广告主类型 | 投放诉求 | 匹配场景 |
|-----------|---------|---------|
| **招工平台**（58同城、BOSS直聘） | 精准触达求职流动人群 | 求职者寄存行李时 |
| **工具/劳保品牌**（得力、3M） | 触达农民工工具使用者 | 农民工寄存工具时 |
| **通信运营商**（移动、联通） | 推广套餐、号卡 | 跨城流动换号需求 |
| **快消品品牌**（康师傅、农夫山泉） | 品牌曝光+渠道下沉 | 高频使用场景 |
| **金融/保险**（微众银行、平安） | 推广普惠金融产品 | 订单确认页 |
| **本地生活服务** | 到达新城市后的生活需求 | 配送完成页 |

**广告定价参考：**

| 广告形式 | 定价模式 | 预估价格 |
|---------|---------|---------|
| 开屏广告 | CPM（千次曝光） | 30-80 元/CPM |
| Banner 广告 | CPC（每次点击） | 0.5-2 元/CPC |
| 信息流广告 | CPA（每次行动） | 5-20 元/CPA |
| 线下仓库广告 | 月租 | 500-2000 元/月/点位 |
| 包裹广告 | 按量计费 | 0.1-0.3 元/件 |

#### 用户端低价策略

| 服务 | 原定价（v1.0） | 新定价（v1.1） | 降幅 |
|------|---------------|---------------|------|
| 暂存费 | 0.5m³/10元/天 | 0.5m³/3-5元/天 | **50-70%** |
| 配送费 | 市场价 | 成本价+微利 | **30-50%** |
| 保价费 | 物品价值 1-3% | 维持不变 | - |
| 新用户优惠 | 无 | 首单免费暂存7天 | 新增 |

> **核心原则**：用户端价格仅覆盖基础运营成本，利润主要来自广告收入。

#### 成本结构

| 成本类型 | 说明 | 优化方向 |
|---------|------|---------|
| 仓储成本 | 自建/合作仓库 | 与现有驿站/物流园合作，轻资产运营 |
| 配送成本 | 同城配送、跨城物流 | 对接快递平台（长途）+ 外卖平台（短途），降低成本 |
| 人力成本 | 客服、运营、仓库管理 | 数字化运营，AI客服辅助 |
| 获客成本 | 线上推广+线下地推 | 低价策略自带传播力，口碑获客 |
| 广告销售成本 | 广告主拓展、销售团队 | 初期自营，后期接入广告联盟（穿山甲、优量汇） |

### 1.5 可行性评估

| 维度 | 评估结果 | 说明 |
|------|---------|------|
| **市场需求** | ⭐⭐⭐⭐⭐ | 1.79亿外出农民工+900万毕业生，痛点明确 |
| **竞争壁垒** | ⭐⭐⭐ | 需建立仓储网络+物流合作，有一定门槛 |
| **商业模式** | ⭐⭐⭐⭐ | 多元收入，轻资产运营可行 |
| **技术难度** | ⭐⭐⭐ | 核心是运营而非技术，小程序+后台即可 |
| **政策风险** | ⭐⭐ | 需关注物流监管、物品安全合规 |

### 1.6 风险与应对

| 风险 | 概率 | 影响 | 应对策略 |
|------|------|------|---------|
| 物品丢失/损坏 | 中 | 高 | 保价机制、仓库监控、合作物流保险 |
| 用户信任不足 | 高 | 高 | 品牌背书、试点城市验证、用户评价体系 |
| 仓储成本过高 | 中 | 中 | 轻资产合作模式、动态调配仓储资源 |
| 获客难度大 | 中 | 高 | 与劳务中介、高校、企业合作，精准触达 |
| 政策监管 | 低 | 高 | 合规运营，与物流企业合作规避风险 |

## 四、PRD 产品需求文档

### 4.1 Executive Summary

#### 问题陈述

外出农民工和求职者在跨城流动时，面临工具/大件物品无法随身携带、目的地地址不确定的困境。现有物流服务均需确定收货地址，无法满足"先寄后定"的场景需求。

#### 解决方案

打造一个面向流动人群的物品暂存与配送一体化平台，用户可将物品提前寄存至目的地附近的仓库，确定具体地址后一键发起配送，实现"地址未定，物品先行"。

#### 商业价值

- 服务 1.79 亿外出农民工 + 900 万毕业生群体
- **广告驱动模式**：低价获取用户 → 广告精准变现 → 规模化盈利
- 轻资产运营模式，与现有仓储/物流合作
- **平台合作**：长途对接快递平台，短途对接外卖平台，降低配送成本

#### 时间线

- MVP 上线：3 个月
- 试点验证：2 个月
- 规模扩张：7 个月

### 4.3 解决方案

#### 产品定位

```
FOR 外出农民工和求职者
WHO 需要跨城携带物品但目的地地址不确定
THE "行囊管家" IS A 物品暂存与配送平台
THAT 让用户提前寄存物品到目的地附近，确定地址后一键配送
UNLIKE 传统物流需要确定收货地址
OUR PRODUCT 支持"地址未定，物品先行"的创新场景
```

#### 核心能力

1. **灵活寄存**：用户只需选择城市/区域，无需确定具体地址
2. **目的地调整**：支持中途修改暂存城市（运输中/暂存中均可），适应流动人群计划变更
3. **透明追踪**：实时查看物品位置和暂存状态
4. **一键配送**：确定地址后，在线发起配送请求
5. **安全保障**：保价服务、仓库监控、合作物流保险
6. **智能配送调度**：长途对接快递平台，短途对接外卖/即时配送平台

#### 平台合作规划

> **v1.1 新增**：明确物流配送的合作策略，长途与短途分别对接不同平台。

**长途配送 —— 对接快递平台**

| 合作方 | 适用场景 | 合作模式 | 优势 |
|--------|---------|---------|------|
| **顺丰速运** | 跨省大件/贵重物品 | API 对接，企业协议价 | 时效快、服务好、品牌信任度高 |
| **德邦快递** | 大件/重货（工具、行李） | API 对接，大件专线 | 大件物流专业、价格合理 |
| **京东物流** | 中短途、时效要求高 | API 对接 | 仓配一体、覆盖广 |
| **中国邮政** | 下沉城市/乡镇末端 | API 对接 | 覆盖最广，触达偏远地区 |

**短途配送 —— 对接外卖/即时配送平台**

| 合作方 | 适用场景 | 合作模式 | 优势 |
|--------|---------|---------|------|
| **美团配送** | 同城配送（仓库→用户地址） | API 对接，运力调度 | 骑手覆盖广、响应快 |
| **达达快送** | 同城即时配送 | API 对接 | 同城配送专业、可预约时段 |
| **饿了么蜂鸟** | 同城小件配送 | API 对接 | 与美团互补，增加运力冗余 |
| **闪送** | 急件、加急配送 | API 对接 | 一对一专送、时效最快 |

**合作策略要点：**

```
                    ┌─────────────────────────────┐
                    │       配送调度引擎            │
                    │  （根据距离/重量/时效智能选择） │
                    └──────────┬──────────────────┘
                               │
                ┌──────────────┼──────────────┐
                │              │              │
         距离 > 50km      10-50km         < 10km
         重量不限        重量 < 20kg     重量 < 10kg
                │              │              │
                ▼              ▼              ▼
         ┌──────────┐  ┌──────────┐  ┌──────────┐
         │ 快递平台  │  │ 达达/闪送 │  │ 美团/蜂鸟 │
         │ 顺丰/德邦 │  │ 即时配送  │  │ 同城配送  │
         └──────────┘  └──────────┘  └──────────┘
```

- **智能路由**：系统根据距离、重量、时效自动选择最优配送渠道
- **多平台冗余**：高峰期自动切换备用平台，保障配送时效
- **统一结算**：用户端一次支付，后台自动与各平台结算
- **价格透明**：配送前展示预估费用，用户确认后发起
