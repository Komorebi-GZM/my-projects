### 校园物品暂存小程序

面向校园场景的物品寄存服务，用户可通过微信小程序下单寄存物品、查看订单状态、发起配送。

**架构**
- 本地开发：Python 3.12 + Flask + SQLite（完整业务逻辑）
- 生产部署：CloudBase 云开发 + 微信小程序

**核心功能**
- 用户下单 / 查询订单 / 配送申请
- 管理后台（订单管理、物品管理）
- CloudBase 数据库 + 云函数一键部署

**快速启动（本地开发模式）**

```bash
pip install flask
cd admin  # 或运行 Flask 入口
python app.py
```
