## 3. 技术细节

### 3.1 StrEnum 迁移

```python
# 之前
from enum import Enum
class Difficulty(str, Enum):
    ...

# 之后  
from enum import StrEnum
class Difficulty(StrEnum):
    ...
```

**优势**: Python 3.12 原生支持，类型检查更严格，IDE 提示更友好。

### 3.2 困毙测试局面设计

```
黑方九宫 (0,3)-(2,5) 全填满士：
  士 将 士
  士 士 士  
  士 士 士

红帅在 (9,3)，被中路士遮挡不构成将军
→ 无将军 + 无合法走子 = 困毙
```
