# Triage 标签定义

## 标签列表

| 标签 | 说明 |
|------|------|
| `needs-triage` | 需要维护者评估 |
| `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | 已完全指定，AFK agent 可执行 |
| `ready-for-human` | 需要人工实现 |
| `wontfix` | 不会处理 |

## 使用说明

- 新创建的 issue 默认标记为 `needs-triage`
- 维护者评估后根据情况更新标签
- `ready-for-agent` 表示 issue 已准备好，可由 AI agent 自动执行
- `ready-for-human` 表示需要人工介入处理
