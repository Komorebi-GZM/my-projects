## 1. 数据概况摘要

| 指标 | 值 |
|------|-----|
| **数据来源** | 印度金融市场新闻（Economic Times 等 7 个来源） |
| **分析单元** | 每条新闻记录（一行 = 一条新闻） |
| **数据维度** | 139,919 行 × 24 列 |
| **时间跨度** | 2016-01-01 ~ 2026-04-15（约 10 年） |
| **内存占用** | 174.7 MB |
| **唯一分类数** | 17 个行业/事件分类（含多标签，原始组合 366 种） |
| **唯一关键词数** | 414 个核心财经关键词 |
| **文本平均长度** | 156.8 字符（title + description 合并后） |
| **缺失值** | 0（全量无缺失） |
| **重复记录** | 0（已去重） |

### 核心字段一览

| 字段 | 类型 | 说明 | 分析价值 |
|------|------|------|----------|
| `date` | datetime | 新闻发布日期 | 时间序列分析、趋势分析 |
| `title` | str | 新闻标题 | 文本挖掘、情感分析 |
| `description` | str | 新闻详情描述 | 文本挖掘、情感分析 |
| `full_text` | str | 标题+描述合并文本 | FinBERT 输入、主题建模 |
| `categories` | str | 行业/事件分类（可多标签） | 分类统计、行业热度 |
| `categories_list` | list | 分类列表形式 | 聚类、交叉分析 |
| `matched_keywords` | str | 核心关键词（可多个） | 关键词频率、主题提取 |
| `keywords_list` | list | 关键词列表形式 | 共现分析、网络图 |
| `relevance_score` | int | 相关度评分（1-11） | 质量筛选、权重计算 |
| `has_negation` | bool | 是否含否定表述 | 情感分析辅助特征 |
| `impact_tier` | str | 影响等级（LOW/MEDIUM/HIGH） | 事件影响力评估 |
| `source_file` | str | 数据来源文件 | 来源偏差分析 |

### 数据缺口说明

| 缺口项 | 说明 | 影响 | 处理方案 |
|--------|------|------|----------|
| **股票 OHLCV 行情数据** | 数据集不含股价数据 | 无法直接做股价预测建模 | **已移除**股价预测相关目标，聚焦文本分析与舆情挖掘 |
| **个股/实体关联字段** | 原始数据集说明中提及 stock 字段但实际未包含 | 无法做个股级别分析 | 聚焦行业/板块级别分析 |

## 2. 业务背景

### 2.1 业务场景

本数据集收录 2016-2026 年印度金融市场每日财经新闻，覆盖宏观经济、行业动态、公司财报、地缘政治等多个维度。数据适用于金融文本挖掘、舆情监控、事件驱动分析等研究场景。

### 2.2 分析动机

- 金融新闻是市场情绪的重要载体，系统化挖掘新闻舆情有助于理解市场行为
- 当前数据集时间跨度长（10年）、覆盖面广（17个分类、414个关键词），具备深度挖掘价值
- 需要建立标准化的金融新闻分析流程，为后续量化研究奠定基础

### 2.3 预期产出

- 全量新闻的可视化统计分析报告
- 新闻情感自动判别体系（FinBERT）
- 金融主题聚类与事件分类体系
- 新闻舆情特征工程流程（可复用）
- 交互式可视化看板

### 目标-4: 新闻舆情特征工程

| 属性 | 内容 |
|------|------|
| **类型** | 数据工程 |
| **优先级** | P1（重要） |
| **SMART 表述** | 基于情感标签、主题分类、关键词统计、时间特征等维度，构建标准化的新闻舆情特征集，输出按日期聚合的日频特征表，为后续建模和策略分析提供可直接使用的特征输入 |
| **关键指标** | 特征数量（目标 ≥ 30 个）、特征覆盖率、特征相关性矩阵 |
| **对应章节** | ch04_feature_engineering |
| **依赖** | ch01 清洗后数据 + ch03 情感标签 + ch03 主题分类 |

**具体分析内容**：
1. **情感特征**: 日均情感得分、情感波动率、正面/负面新闻占比、情感极化指数
2. **主题特征**: 各主题日频新闻计数、主题多样性指数（Shannon 熵）、热门主题切换频率
3. **关键词特征**: Top 关键词日频出现次数、关键词共现网络指标
4. **影响等级特征**: HIGH/MEDIUM/LOW 新闻日频占比、影响等级加权得分
5. **文本统计特征**: 日均文本长度、日新闻总量、否定表述占比
6. **时间特征**: 星期效应、月份效应、工作日/周末差异
7. **交叉特征**: 行业×情感交叉统计、影响等级×情感交叉统计
8. 特征相关性分析与多重共线性检测

**验收标准**：
- [ ] 产出 `ch04_daily_features.csv`（日频聚合特征表）

## 4. 研究问题清单

### 核心研究问题（RQ-Core）

**问题**: 印度金融新闻的舆情特征（情感、主题、影响力）呈现怎样的时空演变规律？能否建立标准化的舆情特征工程流程，为量化金融研究提供可复用的数据支撑？

**背景**: 金融新闻是市场情绪的重要载体，但缺乏系统化的文本挖掘和舆情分析框架。本数据集覆盖 10 年、17 个行业、414 个关键词，具备深度挖掘条件。

**数据支持**: 139,919 条新闻的 full_text、categories、matched_keywords、impact_tier、relevance_score、has_negation 等字段。

**预期产出**: 情感标签体系、主题分类体系、日频舆情特征表、事件影响力评估体系、可视化看板。

## 5. 分析范围与约束

### 5.1 分析范围

| 维度 | 范围 | 说明 |
|------|------|------|
| **时间范围** | 2016-01-01 ~ 2026-04-15 | 全量数据，不做时间截断 |
| **地理范围** | 印度金融市场 | 数据来源为印度财经媒体 |
| **行业范围** | 17 个分类 | macro_government, stock_specific, geopolitical 等 |
| **文本范围** | title + description 合并文本 | 平均 156.8 字符/条 |
| **分析粒度** | 日频聚合为主 | 部分分析支持月度/季度 |

### 5.2 约束条件

| 约束 | 说明 | 影响 |
|------|------|------|
| **无股价数据** | 数据集不含 OHLCV 行情数据 | 移除股价预测目标，聚焦文本分析 |
| **无个股标识** | 数据集不含 stock 字段 | 聚焦行业/板块级别，不做个股分析 |
| **英文文本** | 新闻为英文 | NLP 工具需支持英文（FinBERT 原生支持） |
| **数据偏差** | 2021 年数据量骤降（仅 3,548 条） | 时序分析需注意数据稀疏区间 |
| **计算资源** | 单机环境 | FinBERT 推理需考虑 GPU 可用性，备选 CPU 批量推理 |

### 5.3 技术选型

| 任务 | 推荐工具/方法 | 备选方案 |
|------|--------------|----------|
| 情感分析 | FinBERT (ProsusAI/finbert) | VADER, TextBlob |
| 主题建模 | BERTopic | LDA (gensim) |
| 文本预处理 | NLTK / spaCy | sklearn CountVectorizer |
| 可视化 | matplotlib + seaborn + plotly | pyecharts |
| 特征工程 | pandas + sklearn | featuretools |
| 看板 | Streamlit | Dash |
