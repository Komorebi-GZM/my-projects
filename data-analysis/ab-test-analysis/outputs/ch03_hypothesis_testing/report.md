# ch03 假设检验与效应量 - 执行报告

## 一、背景

本章节对 A/B 测试结果进行严格的统计检验，评估实验组点击率提升是否具有统计显著性，以及效应量是否具有实际业务意义。采用比例 Z 检验（单尾）进行假设检验，并计算 Cohen's h 效应量、统计功效和最小可检测效应（MDE）。

## 二、分析方法

### 2.1 假设检验

- **原假设 H₀**：p_exp ≤ p_con（实验组点击率不高于对照组）
- **备择假设 H₁**：p_exp > p_con（实验组点击率高于对照组）
- **显著性水平**：α = 0.05

### 2.2 计算方法

| 分析内容 | 方法 | 公式 |
|----------|------|------|
| Z 统计量 | 比例 Z 检验 | Z = (p_exp - p_con) / SE_pooled |
| 合并标准误 | Pooled SE | SE = √(p̂(1-p̂)(1/n₁+1/n₂)) |
| 效应量 | Cohen's h | h = 2(arcsin√p_exp - arcsin√p_con) |
| 统计功效 | statsmodels | zt_ind_solve_power |

### 2.3 效应量解释标准

| Cohen's h | 解释 |
|-----------|------|
| < 0.2 | 可忽略（negligible） |
| 0.2 ~ 0.5 | 小效应（small） |
| 0.5 ~ 0.8 | 中等效应（medium） |
| ≥ 0.8 | 大效应（large） |

## 三、分析发现

### 3.1 假设检验结果

| 指标 | 数值 | 解释 |
|------|------|------|
| Z 统计量 | ~4.5 | 远大于临界值 1.645 |
| p 值 | < 0.001 | 高度显著 |
| 结论 | 拒绝 H₀ | 实验组点击率显著高于对照组 |

### 3.2 效应量分析

| 指标 | 数值 | 解释 |
|------|------|------|
| Cohen's h | ~0.06 | 可忽略效应 |
| 效应量解释 | negligible | 统计显著但实际效应很小 |

### 3.3 统计功效

| 指标 | 数值 | 评估 |
|------|------|------|
| 统计功效 | > 0.99 | 功效充足（> 80%） |
| MDE | ~1.2% | 最小可检测效应 |

### 3.4 CTR 差异的 95% 置信区间

| 指标 | 数值 |
|------|------|
| 差异点估计 | ~2.0% |
| 95% CI 下限 | ~1.2% |
| 95% CI 上限 | ~2.8% |

## 四、小结

1. **统计显著**：p 值 < 0.001，实验组点击率显著高于对照组
2. **效应量小**：Cohen's h ≈ 0.06，属于可忽略效应，统计显著但实际业务意义有限
3. **功效充足**：统计功效 > 99%，实验设计足够敏感
4. **置信区间不包含 0**：差异的 95% CI 为 [1.2%, 2.8%]，确认正向提升

### 决策建议

根据严格决策框架（p值 + 效应量 + 功效）：
- p < 0.05 ✅
- Cohen's h < 0.2 ⚠️
- 功效 ≥ 0.8 ✅

**建议**：灰度发布，持续观察长期效果
