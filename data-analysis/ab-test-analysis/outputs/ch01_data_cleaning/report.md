# ch01 数据清洗 - 执行报告

## 一、背景

本章节对 A/B 测试原始数据进行质量检查与清洗，确保数据符合后续分析要求。原始数据包含 20,000 条用户点击记录，涵盖对照组（con）和实验组（exp）两组用户在 2024-01-01 至 2024-01-07 期间的点击行为。

数据质量是后续统计分析的基础，需要检查缺失值、重复值、数据类型一致性等问题，确保分析结论的可靠性。

## 二、分析方法

采用以下方法进行数据清洗：

1. **缺失值检查**：使用 `pandas.isnull()` 统计各字段的缺失值数量和比例
2. **重复值检查**：使用 `pandas.duplicated()` 识别完全重复的记录
3. **数据类型验证**：
   - 将 `timestamp` 转换为 datetime 类型
   - 验证 `click` 取值仅为 0/1
   - 验证 `group` 取值仅为 con/exp
   - 验证 `user_id` 唯一性
4. **描述性统计**：按分组统计用户数、点击数、点击率
