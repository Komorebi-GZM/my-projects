# AB_Test_Analysis 分析目标文档

> 项目：AB_Test_Analysis — A/B测试点击率数据分析
> 生成时间：2026-05-07
> 数据概况：20,000 名用户，4 个字段（user_id, click, group, timestamp）

---

## 1. 业务背景

### 1.1 数据来源与场景

在互联网产品迭代中，页面设计、文案、功能布局等变化对用户点击行为的影响是衡量优化效果的关键指标。传统的"全量上线"方式风险高，而 **A/B 测试（随机对照实验）** 能够通过小流量实验，量化评估新版本相比旧版本在核心指标（如点击率、转化率）上的提升是否具有统计显著性，从而指导产品决策。

本数据集记录了 **20,000 名用户** 在某个线上产品中的一次 A/B 测试实验数据。

### 1.2 分析单元

每一行代表 **一个用户的一次实验观测**，包含：
- `user_id`：用户唯一标识
- `click`：二值指标（1 = 点击，0 = 未点击）
- `group`：分组标识（`con` = 对照组/旧版本，`exp` = 实验组/新版本）
- `timestamp`：实验时间戳（2024-01-01 ~ 2024-01-07）

### 1.3 关键业务指标

| 指标 | 定义 | 计算方式 |
|------|------|----------|
| **点击率 (CTR)** | 用户点击比例 | 点击数 / 总用户数 × 100% |
| **绝对提升** | 实验组 CTR - 对照组 CTR | 百分点差异 |
| **相对提升** | 绝对提升 / 对照组 CTR | 百分比变化 |
| **Cohen's h** | 两比例效应量 | 衡量差异的实际业务意义 |

### 1.4 数据现状（清洗后）

| 分组 | 用户数 | 点击数 | 点击率 |
|------|--------|--------|--------|
| 对照组 (con) | 10,012 | 812 | 8.11% |
| 实验组 (exp) | 9,988 | 973 | 9.74% |
| **差异** | — | +161 | **+1.63 pp** |

初步观察：实验组点击率高于对照组，需通过统计检验确认是否显著。

## 2. 分析动机

### 2.1 为什么现在做这个分析？

产品团队完成了新版本页面的 A/B 测试实验，数据已收集完毕（7天），需要通过规范的统计分析流程评估实验效果，为下一步产品决策提供数据支撑。

### 2.2 期望的分析产出

1. **分组核心指标对比表**（CTR、绝对提升、相对提升）
2. **带 95% 置信区间的可视化图表**（条形图/点图）
3. **假设检验报告**（Z 统计量、p 值、效应量 Cohen's h、置信区间）
4. **统计功效与 MDE 分析**（验证实验灵敏度是否充足）
5. **时间维度趋势分析**（按天/小时的点击率变化趋势）
6. **明确的业务决策建议**（全量上线 / 灰度发布 / 进一步实验 / 放弃）

### 2.3 分析结果的使用方式

分析报告将直接提交给产品决策团队，作为是否将新版本全量上线的核心依据。

## 3. 约束条件

| 约束项 | 说明 |
|--------|------|
| 显著性水平 | α = 0.05 |
| 检验类型 | 单尾检验（H₁: 实验组 > 对照组） |
| 效应量评估标准 | Cohen's h: 小(0.2) / 中(0.5) / 大(0.8) |
| 时间范围 | 2024-01-01 ~ 2024-01-07（7天） |
| 分组 | 对照组(con) vs 实验组(exp) |
| 关注维度 | 整体 + 时间趋势（按天/按小时） |

## 4. SMART 分析目标

### 目标 1：计算分组核心指标（描述性分析）

- **S**pecific：计算对照组和实验组的点击率 CTR，以及绝对提升和相对提升
- **M**easurable：输出数值型指标表，精确到小数点后 2 位
- **A**chievable：基于清洗后的 20,000 条数据，直接计算
- **R**elevant：CTR 是衡量页面优化效果的核心业务指标
- **T**ime-bound：当前数据集范围（2024-01-01 ~ 2024-01-07）

### 目标 2：可视化对比（描述性分析）

- **S**pecific：绘制带 95% 置信区间的条形图/点图，展示两组 CTR 差异
- **M**easurable：图表包含误差线、数值标注、图例
- **A**chievable：使用 matplotlib/seaborn 绘制
- **R**elevant：可视化直观展示实验效果，便于非技术人员理解
- **T**ime-bound：当前数据集范围

### 目标 3：假设检验与效应量评估（推断性分析）

- **S**pecific：
  - 建立假设 H₀/H₁，执行比例 Z 检验
  - 计算 p 值、Z 统计量
  - 计算 Cohen's h 效应量及 95% 置信区间
  - 执行统计功效分析，计算 MDE（最小可检测效应）
- **M**easurable：p 值（精确到 4 位小数）、Cohen's h、功效值（≥0.8 为合格）
- **A**chievable：使用 statsmodels/scipy 实现
- **R**elevant：统计检验是 A/B 测试结论的科学基础
- **T**ime-bound：当前数据集范围

### 目标 4：时间维度趋势分析（诊断性分析）

- **S**pecific：按天/按小时拆分点击率，观察两组随时间的变化趋势
- **M**easurable：时间序列折线图 + 按天的分组统计表
- **A**chievable：基于 timestamp 字段分组聚合
- **R**elevant：检测是否存在新奇效应（Novelty Effect）或时间异质性
- **T**ime-bound：2024-01-01 ~ 2024-01-07

### 目标 5：结论与决策建议（规范性分析）

- **S**pecific：综合 p 值、效应量、置信区间、统计功效，给出明确业务建议
- **M**easurable：决策结论 + 理由清单 + 风险提示
- **A**chievable：基于前述分析结果综合判断
- **R**elevant：直接服务于产品全量上线决策
- **T**ime-bound：当前分析周期

## 5. 研究问题清单

### 核心研究问题（RQ-Core）

**问题**：新版本页面（实验组）相比旧版本页面（对照组）是否显著提升了用户点击率？提升幅度是否具有实际业务意义？

**背景**：产品团队通过 A/B 测试评估新版本页面的效果，需要科学的统计证据支持全量上线决策。

**数据支持**：20,000 名用户的分组实验数据（user_id, click, group, timestamp）

**预期产出**：统计检验报告 + 效应量评估 + 明确的业务决策建议

---

### 子研究问题

#### RQ-Sub-1：两组的点击率分别是多少？差异有多大？

- **与核心问题关系**：提供描述性基础数据
- **分析方法**：分组聚合计算、描述统计
- **所需数据**：click, group
- **产出形式**：分组指标对比表

#### RQ-Sub-2：两组点击率的差异是否具有统计显著性？

- **与核心问题关系**：核心统计推断
- **分析方法**：比例 Z 检验（单尾），α = 0.05
- **所需数据**：click, group
- **产出形式**：Z 统计量、p 值、95% 置信区间

#### RQ-Sub-3：差异的实际业务意义有多大？

- **与核心问题关系**：评估效应量，避免"统计显著但业务无意义"
- **分析方法**：Cohen's h 效应量计算 + 解释
- **所需数据**：两组点击率
- **产出形式**：Cohen's h 值 + 业务解释

#### RQ-Sub-4：实验的统计功效是否充足？MDE 是多少？

- **与核心问题关系**：验证实验设计的灵敏度，避免假阴性
- **分析方法**：统计功效分析（基于实际样本量和观测到的效应量）
- **所需数据**：样本量、两组比例、α、效应量
- **产出形式**：统计功效值、MDE

#### RQ-Sub-5：点击率随时间如何变化？是否存在时间异质性？

- **与核心问题关系**：检测新奇效应或时间趋势，增强结论稳健性
- **分析方法**：按天/按小时分组聚合，绘制时间序列折线图
- **所需数据**：click, group, timestamp
- **产出形式**：时间趋势折线图 + 按天统计表

#### RQ-Sub-6：基于以上分析，应给出什么业务决策建议？

- **与核心问题关系**：最终决策输出
- **分析方法**：综合 p 值、效应量、功效、时间趋势进行判断
- **所需数据**：前述所有分析结果
- **产出形式**：决策建议（全量上线 / 灰度发布 / 进一步实验 / 放弃）+ 理由

## 6. 决策框架

### 严格决策矩阵

| 条件 | 决策建议 |
|------|----------|
| p < 0.05 且 Cohen's h ≥ 0.2 且 功效 ≥ 0.8 | ✅ **全量上线** |
| p < 0.05 且 Cohen's h < 0.2 | ⚠️ **灰度发布**（统计显著但效应小） |
| p ≥ 0.05 且 功效 < 0.8 | 🔄 **需要进一步实验**（可能假阴性） |
| p ≥ 0.05 且 功效 ≥ 0.8 | ❌ **放弃该版本**（功效充足仍不显著） |

### 效应量解读标准（Cohen's h）

| 范围 | 含义 |
|------|------|
| h < 0.2 | 可忽略（negligible） |
| 0.2 ≤ h < 0.5 | 小效应（small） |
| 0.5 ≤ h < 0.8 | 中等效应（medium） |
| h ≥ 0.8 | 大效应（large） |
