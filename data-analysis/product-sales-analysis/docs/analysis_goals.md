# 产品销售数据分析 — 分析目标文档

> 项目：Product_Sales_Analysis
> 生成日期：2026-05-08
> 数据概况：1000 条销售记录，8 个字段，时间跨度 2025-01-02 至 2026-05-16

---

## 一、业务背景

### 1.1 数据来源与场景

本数据集来自零售/电商产品销售场景，每行代表一笔产品销售交易记录。数据涵盖 6 个产品类别（Beauty, Books, Electronics, Fashion, Home, Sports）、24 种产品、5 个城市（Islamabad, Karachi, Lahore, Peshawar, Quetta）。

### 1.2 数据字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| Product_ID | int | 产品唯一标识（1001-2000） |
| Product_Name | str | 产品名称（24 种） |
| Category | str | 产品类别（6 类） |
| Price_USD | int | 单价（美元，范围 10-500） |
| Quantity_Sold | int | 销售数量（范围 1-10） |
| Total_Sales_USD | int | 销售总额（美元，范围 10-5000） |
| Order_Date | date | 订单日期 |
| Customer_City | str | 客户所在城市（5 个） |

### 1.3 关键业务指标

| 指标 | 计算方式 | 业务含义 |
|------|---------|---------|
| 总销售额 | Σ Total_Sales_USD | 整体营收规模 |
| 平均客单价 | Σ Total_Sales_USD / 记录数 | 单笔交易贡献 |
| 品类销售额占比 | 品类销售额 / 总销售额 | 品类贡献度 |
| 城市销售额占比 | 城市销售额 / 总销售额 | 区域贡献度 |
| 月度销售趋势 | 按月聚合 Total_Sales_USD | 销售走势 |
| 价格弹性系数 | 销量变化率 / 价格变化率 | 价格敏感度 |

### 1.4 当前业务痛点

- 各品类/城市的销售表现差异不明，缺乏数据驱动的资源分配依据
- 销售趋势是否存在季节性或周期性规律未知
- 定价策略缺乏数据支撑，不知道价格调整对销量的实际影响
- 缺乏整体业务洞察和可落地的运营建议

### 1.5 数据局限性

| 局限 | 影响 | 应对策略 |
|------|------|---------|
| 无 Customer_ID | 无法做 RFM 用户分层和购物篮分析 | 聚焦产品/品类/城市/时间维度 |
| 无成本/利润字段 | 无法计算利润率 | 以销售额和销量为核心指标 |
| Price_USD 为整数 | 可能非真实定价精度 | 弹性分析结果需谨慎解读 |
| 数据量 1000 条 | 统计检验效力有限 | 侧重描述性分析和趋势判断 |

## 三、SMART 分析目标

### 目标 1：销售现状全景描述

| 维度 | 内容 |
|------|------|
| **S**pecific | 对 1000 条销售记录进行多维度描述性统计分析，覆盖品类、城市、产品、时间四个维度 |
| **M**easurable | 输出：描述统计表、6 类品类分布图、5 城市对比图、月度趋势图、Top/Bottom 产品排名 |
| **A**chievable | 数据完整无缺失，8 个字段均可直接用于聚合和可视化 |
| **R**elevant | 直接回答"当前销售表现如何"的核心业务问题 |
| **T**ime-bound | 分析覆盖 2025-01-02 至 2026-05-16 全部数据 |

### 目标 2：销售趋势预测

| 维度 | 内容 |
|------|------|
| **S**pecific | 按月聚合总销售额，使用统计方法（移动平均、指数平滑）预测未来 3 个月销售趋势 |
| **M**easurable | 输出：月度趋势图、预测曲线、预测区间、模型误差指标（MAE/RMSE） |
| **A**chievable | 约 17 个月的数据量足够进行基础时序预测 |
| **R**elevant | 为库存规划和销售目标设定提供数据依据 |
| **T**ime-bound | 历史数据 2025-01 至 2026-05，预测 2026-06 至 2026-08 |

### 目标 3：价格弹性分析

| 维度 | 内容 |
|------|------|
| **S**pecific | 分析同一产品在不同价格水平下的销量变化，计算品类级价格弹性系数 |
| **M**easurable | 输出：价格-销量散点图、品类弹性系数表、弹性分类（弹性/刚性/单位弹性） |
| **A**chievable | 数据包含 Price_USD 和 Quantity_Sold，可按产品/品类分组分析 |
| **R**elevant | 为定价策略优化提供量化依据 |
| **T**ime-bound | 使用全量数据，不设时间窗口限制 |

### 目标 4：业务策略建议

| 维度 | 内容 |
|------|------|
| **S**pecific | 基于前三个目标的发现，输出结构化的业务建议报告，涵盖品类策略、区域策略、定价策略 |
| **M**easurable | 输出：结论与建议报告（report.md），每条建议需有数据支撑 |
| **A**chievable | 基于前序分析结果归纳总结，不涉及额外数据需求 |
| **R**elevant | 直接服务于业务决策，是整个分析的最终价值交付 |
| **T**ime-bound | 在前三章分析完成后立即输出 |

## 四、研究问题清单

### 核心研究问题（RQ-Core）

**问题**: 产品销售的整体表现如何？哪些品类、城市和产品是增长引擎，哪些需要关注？

**背景**: 管理层需要了解当前业务全景，以做出资源分配和战略调整决策。

**数据支持**: Total_Sales_USD, Quantity_Sold, Category, Customer_City, Order_Date

**预期产出**: 多维度统计表 + 可视化图表 + 关键发现摘要

---

### 子研究问题

#### RQ-Sub-1: 各品类的销售贡献和增长趋势如何？

- **与核心问题关系**: 品类维度分析
- **分析方法**: 分组聚合、占比计算、月度趋势折线图
- **所需数据**: Category, Total_Sales_USD, Quantity_Sold, Order_Date
- **产出形式**: 品类销售额/销量汇总表、品类占比饼图、品类月度趋势图

#### RQ-Sub-2: 各城市的销售表现有何差异？

- **与核心问题关系**: 区域维度分析
- **分析方法**: 分组聚合、城市对比柱状图、城市-品类交叉分析
- **所需数据**: Customer_City, Category, Total_Sales_USD, Quantity_Sold
- **产出形式**: 城市销售额排名、城市-品类热力图、城市月度趋势

#### RQ-Sub-3: Top/Bottom 产品有哪些？产品间的销售差异有多大？

- **与核心问题关系**: 产品维度分析
- **分析方法**: 排名统计、箱线图、帕累托分析（80/20法则）
- **所需数据**: Product_Name, Total_Sales_USD, Quantity_Sold, Price_USD
- **产出形式**: 产品销售排名表、帕累托图、产品价格-销量散点图

#### RQ-Sub-4: 销售额是否存在时间趋势或季节性？

- **与核心问题关系**: 时间维度分析 + 预测基础
- **分析方法**: 月度聚合、趋势分解、移动平均平滑
- **所需数据**: Order_Date, Total_Sales_USD
- **产出形式**: 月度趋势图、移动平均线、季节性分解图

#### RQ-Sub-5: 未来 3 个月的销售额预测是多少？

- **与核心问题关系**: 预测性分析
- **分析方法**: 简单移动平均（SMA）、指数平滑（SES/Holt）、误差评估（MAE/RMSE）
- **所需数据**: Order_Date, Total_Sales_USD（按月聚合）
- **产出形式**: 预测曲线图、预测值表、模型误差对比表

#### RQ-Sub-6: 价格变动对销量有什么影响？各品类的价格弹性如何？

- **与核心问题关系**: 定价策略分析
- **分析方法**: 价格-销量相关分析、按品类分组回归、弹性系数计算
- **所需数据**: Price_USD, Quantity_Sold, Category, Product_Name
- **产出形式**: 价格-销量散点图、品类弹性系数表、弹性分类标签

#### RQ-Sub-7: 基于以上发现，应采取哪些业务行动？

- **与核心问题关系**: 结论与建议
- **分析方法**: 综合前序分析结果，归纳关键发现，提出针对性建议
- **所需数据**: 前序所有分析结果
- **产出形式**: 结构化建议报告（按品类/区域/定价分类）

## 六、决策框架

### 分析方法选择依据

| 分析需求 | 选择方法 | 选择理由 |
|---------|---------|---------|
| 品类/城市对比 | 分组聚合 + 柱状图 | 直观展示差异，适合混合受众 |
| 时间趋势 | 月度折线图 + 移动平均 | 消除噪声，突出趋势 |
| 销售预测 | SMA + 指数平滑 | 数据量有限，统计方法更稳健可解释 |
| 价格弹性 | 分组相关分析 + 回归 | 量化价格与销量关系 |
| 综合建议 | 结构化归纳 | 面向混合受众，兼顾洞察和方法 |

### 质量标准

| 标准 | 要求 |
|------|------|
| 数据完整性 | 清洗后无缺失值、无重复值、数据类型正确 |
| 可视化规范 | 统一配色、中文标签、图例完整、DPI≥150 |
| 预测精度 | MAE 不超过月均销售额的 20% |
| 报告规范 | 每章输出 report.md，遵循四段框架（背景/方法/发现/小结） |
| 建议可落地 | 每条建议需有数据支撑，明确优先级和预期效果 |
