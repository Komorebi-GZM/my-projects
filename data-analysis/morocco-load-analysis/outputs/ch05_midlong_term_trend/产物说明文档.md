# Prompt-05 中长期趋势分析 — 产物说明文档

> **章节**: 第五章 月度/季度中长期用电趋势分析
> **输入数据**: `ch01_cleaned_data.csv`（4城市10分钟粒度kW统一数据集）
> **分析方法**: 多粒度重采样 + 时序分解（STL/移动平均）+ 季节性强度 + 同比/环比
> **产物总数**: 27个文件（7个CSV + 20个PNG）

---

## 一、数据产物说明

### 1. `ch05_daily_monthly_quarterly.csv` — 多粒度聚合数据

| 项目 | 说明 |
|------|------|
| **是什么** | 4城市月度平均负荷汇总表，含等权均值和总量求和两种口径 |
| **为什么需要** | 后续所有分析步骤（STL分解、季节性强度、同比环比等）的数据基础；将10分钟高频数据平滑为月度，消除短时随机波动 |
| **列结构** | `DateTime`(索引), `{城市}_monthly_avg_kw`(等权均值), `{城市}_monthly_sum_kw`(总量求和) |
| **行数** | 21行（2022-09 ~ 2024-05），Marrakech从2023-01开始 |
| **口径说明** | 等权均值 = 各zone列算术平均（主分析口径）；总量求和 = 各zone列直接求和（参考口径） |

### 2. `ch05_stl_components_{city}.csv` — 时序分解数据（×4）

| 项目 | 说明 |
|------|------|
| **是什么** | 每个城市月度负荷的分解结果，包含原始序列、趋势项、季节项、残差项 |
| **为什么需要** | 为趋势对比图（Step 5.3）和季节性强度计算（Step 5.4）提供数据输入 |
| **列结构** | `DateTime`(索引), `observed`(原始月度均值), `trend`(趋势项), `seasonal`(季节项), `residual`(残差项), `method`(分解方法) |
| **分解方法** | 4城市月度数据均不足24个月，全部使用 **3月中心移动平均（Moving Average）** 替代STL |
| **城市文件** | `ch05_stl_components_laayoune.csv`, `_boujdour.csv`, `_foum_eloued.csv`, `_marrakech.csv` |

### 3. `ch05_seasonal_strength.csv` — 季节性强度表

| 项目 | 说明 |
|------|------|
| **是什么** | 4城市季节性强度指标 F_s 汇总表 |
| **为什么需要** | 量化季节因素对负荷变化的影响程度；为 Prompt-06（跨国对比）和 Prompt-07（配电网优化）提供季节性约束依据 |
| **列结构** | `city`(城市), `var_seasonal`(季节方差), `var_residual`(残差方差), `seasonal_strength`(F_s值), `strength_level`(强度等级), `decomposition_method`(分解方法) |
| **F_s 公式** | F_s = max(0, 1 - Var(残差) / Var(季节+残差))，值域 [0, 1] |
| **分级标准** | >0.7 强季节性 / 0.4~0.7 中等季节性 / ≤0.4 弱季节性 |
| **本次结果** | Laayoune 0.643 / Boujdour 0.586 / Foum eloued 0.601 / Marrakech 0.582（均为中等季节性） |

### 4. `ch05_yoy_mom_analysis.csv` — 同比/环比分析表

| 项目 | 说明 |
|------|------|
| **是什么** | 4城市月度负荷的环比（MoM）和同比（YoY）变化率汇总表 |
| **为什么需要** | 量化负荷的月度短期波动和年度真实增长趋势；为 Prompt-06（跨国对比的负荷变化维度）提供数据 |
| **列结构** | `DateTime`(索引), `monthly_avg_kw`(月度均值kW), `mom_change_pct`(环比变化率%), `yoy_change_pct`(同比变化率%), `city`(城市) |
| **NaN说明** | 环比第1个月为NaN（无上月）；同比前12个月为NaN（无去年同期）；Marrakech同比几乎全NaN（仅13个月数据） |

## 四、关键发现摘要

| 城市 | 月度数据量 | 趋势方向 | 趋势变化率 | 季节性强度 F_s | 强度等级 |
|------|-----------|---------|-----------|---------------|---------|
| Laayoune | 21个月 | 下降 | -7.6% | 0.643 | 中等 |
| Boujdour | 21个月 | 下降 | -15.2% | 0.586 | 中等 |
| Foum eloued | 21个月 | 下降 | -45.5% | 0.601 | 中等 |
| Marrakech | 13个月 | 上升 | +20.3% | 0.582 | 中等 |

**说明**：
- 所有城市季节性强度均为中等（0.4 < F_s < 0.7），季节性波动占负荷变化的主导地位
- 撒哈拉三城市（Laayoune/Boujdour/Foum eloued）负荷呈下降趋势，可能与产业转移或能效提升有关
- Marrakech负荷呈上升趋势，且绝对值远高于其他三城市（因计量单位和产业结构差异）
- 所有城市均使用移动平均替代STL分解（数据不足24个月），结果仅供参考

## 六、数据局限性说明

1. **数据长度不足**：4城市月度数据均不足24个月，无法使用标准STL分解，改用3月移动平均替代，趋势和季节性估计精度受限
2. **Marrakech同比无效**：Marrakech仅13个月数据，同比分析无统计意义（仅1个有效值）
3. **等权均值假设**：综合负荷计算假设各zone权重相等，实际各zone可能覆盖不同数量用户
4. **撒哈拉气候特殊性**：Laayoune/Boujdour/Foum eloued位于撒哈拉地区，夏季极端高温可能导致独特负荷模式（如午间休眠）
5. **计量单位差异**：Marrakech数据为有功功率（kW），其他三城市由电流换算而来，可能存在换算误差
