# 中摩典型城市用电特征跨国差异性分析报告

> **数据来源**: 摩洛哥侧数据来自智能电表实测（10分钟粒度），中国侧数据来自国家统计局/国家电网公开数据（手动录入）
> **对标城市**: 西宁（青海）、银川（宁夏）、酒泉（甘肃）

## 1. 对标城市概况

### 1.1 摩洛哥侧城市
- **Laayoune**（拉尤恩）、**Boujdour**（布杰杜尔）、**Foum eloued**（富姆埃卢德）、**Marrakech**（马拉喀什）
- 气候：干旱/半干旱气候，夏季炎热（均温>25C），冬季温和
- 产业：旅游、渔业、轻工业为主
- 数据来源：智能电表10分钟粒度负荷数据（2022-09 ~ 2024-05）

### 1.2 中国侧对标城市
- **西宁**（青海）：高原干旱气候，人口248万，GDP 1680亿元，年均温6.0C，年降水380mm，主要产业：旅游、轻工业、新能源
- **银川**（宁夏）：温带大陆性气候，人口289万，GDP 2536亿元，年均温9.0C，年降水200mm，主要产业：能源化工、农业、食品加工
- **酒泉**（甘肃）：温带大陆性气候，人口113万，GDP 820亿元，年均温8.0C，年降水80mm，主要产业：新能源、旅游、航空航天

## 3. 差异性归因分析

### 3.1 气候因素

北非摩洛哥城市受撒哈拉热浪影响，夏季（6-8月）制冷负荷显著增加，导致负荷呈现明显的单峰季节性模式。中国西北城市受大陆性气候影响，冬夏温差大，采暖（冬季）和制冷（夏季）需求并存，但冬季采暖主要依赖集中供暖（非电力），因此电力负荷的季节性可能不如摩洛哥显著。

### 3.2 产业结构

摩洛哥城市以旅游和轻工业为主，工业基荷较低，负荷受旅游季节性影响较大。中国西北城市能源化工占比较高，工业基荷稳定，负荷率较高，季节性相对较弱。

- 中国对标城市平均工业用电占比: 75.1%
- 青海: 72.1%, 宁夏: 81.5%, 甘肃: 71.8%

### 3.3 电气化水平

摩洛哥人均用电量远低于中国对标城市，反映电气化程度和经济发展阶段的差距。随着摩洛哥经济发展和电气化推进，预期负荷将呈持续增长趋势。

- 中国对标城市估算人均用电量: 13620 kWh/人/年
- 摩洛哥侧缺乏人口数据，无法计算人均用电量

### 3.4 建筑能效

摩洛哥建筑保温性能较差，导致制冷/采暖能耗占比高，负荷的季节性波动较大。中国西北城市近年来大力推进建筑节能改造，建筑能效提升有助于降低季节性负荷波动。

## 4. 结论与建议

1. 摩洛哥城市负荷的季节性强于中国对标城市，主要受夏季制冷需求驱动
2. 中国对标城市的负荷率普遍高于摩洛哥城市，反映工业基荷的稳定作用
3. 气候差异是中摩用电模式差异的首要驱动因素
4. 建议摩洛哥推进建筑节能改造和可再生能源利用，降低季节性负荷波动
5. 中国西北城市的新能源高占比（青海85%）为摩洛哥提供了可借鉴的发展路径

## 5. 数据来源与可比性局限性说明

### 5.1 数据来源

- **摩洛哥侧**: 智能电表实测数据（10分钟粒度，zone级，2022-09 ~ 2024-05）
- **中国侧**: 国家统计局2023年统计公报（省级，手动录入）、国家电网2023年社会责任报告（手动录入）
- **气象数据**: OpenMeteo API / 手动录入（基于公开气象资料的正弦曲线模拟）

### 5.2 可比性分析

**可跨尺度对比的维度**（比值/强度类指标，不受量级影响）:

| 维度 | 摩洛哥侧 | 中国侧 | 可比性 |
|------|----------|--------|--------|
| 负荷率 (mean/max) | zone级计算 | 省级估算 | **可比** (比值) |
| 季节性强度 Fs | STL分解 | 无直接数据 | **部分可比** |
| 变异系数 CV | zone级计算 | 无直接数据 | **部分可比** |
| 峰谷差比 | zone级计算 | 省级数据 | **可比** (比值) |

**不可跨尺度对比的维度**（绝对值，量级差 6~8 个数量级）:

| 维度 | 摩洛哥侧 | 中国侧 | 不可比原因 |
|------|----------|--------|------------|
| 人均用电量 | 无人口数据 | 省级估算 | 摩洛哥无数据 |
| 总负荷 | kW级 (zone) | GW级 (省级) | 量级差6-8个数量级 |
| GDP | 无数据 | 省级数据 | 摩洛哥无数据 |

### 5.3 数据口径差异

- **空间粒度**: 摩洛哥 = 城市 zone 级（智能电表，几十到几百 kW），中国 = 省级（统计公报，亿 kWh 级）
- **时间粒度**: 摩洛哥 = 10分钟/30分钟，中国 = 年度/月度统计
- **负荷率定义**: 摩洛哥 = mean/max（微观 zone 级），中国 = (max+min)/2/max（宏观省级）
- **估算方法**: 中国侧城市级数据通过人口比例法从省级数据估算，假设用电量与人口成正比，实际偏差可能较大
